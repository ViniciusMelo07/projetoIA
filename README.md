# Core Clothing - Sistema de Recomendação com IA

## 📌 Descrição
Projeto de Inteligência Artificial aplicado ao e-commerce Core Clothing, com foco em recomendação de produtos baseada no comportamento do usuário.

## 🧠 Tecnologias
- Python (pandas, matplotlib)
- Inteligência Artificial
- Next.js (integração futura)

## 📊 Dataset
Dados reais simulados do comportamento do usuário:
- visualizações
- compras
- categorias acessadas

## 📁 Estrutura
- data → dataset
- notebooks → análise exploratória
- src → código Python
- docs → relatório
